#  Descripción

Este proyecto contiene ejercicios realizados en Java sobre:
- Manejo de excepciones

- Conversión de datos
- Casting de variables y objetos

El objetivo es aprender a controlar errores en los programas y trabajar correctamente con diferentes tipos de datos.
